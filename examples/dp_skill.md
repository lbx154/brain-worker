# DP 解题 Skill（速查手册）

## 五步流程

1. **定状态**：找到「决策到第 i 步时，描述局面所需的最少变量」→ `dp[i][...]`
2. **定转移**：枚举「最后一步」的所有选择，写出子问题间的递推关系
3. **定初始**：最小子问题的值（通常 `dp[0]=0` 或 `dp[0][0]=0`）
4. **定顺序**：保证计算 `dp[i]` 时，所依赖的状态已算完
5. **定答案**：明确从哪个状态读取最终结果（`dp[n]` / `max(dp[n][*])` / 全局维护）

---

## 模式一：线性 DP（序列/数组问题）

**状态**：`dp[i]` = 考虑前 i 个元素时的最优值

**典型**：最长递增子序列 (LIS)

```
dp[i] = max(dp[j] + 1)  for all j < i where a[j] < a[i]
```

**适用**：打家劫舍、最大子数组、编辑距离等序列问题

---

## 模式二：背包 DP（选/不选 + 容量限制）

**状态**：`dp[i][w]` = 前 i 个物品、容量为 w 时的最优值

**0-1 背包**：

```
dp[i][w] = max(dp[i-1][w], dp[i-1][w-c[i]] + v[i])  if w >= c[i]
```

**滚动优化**：一维逆序枚举 `for w in W..c[i]: dp[w] = max(dp[w], dp[w-c[i]]+v[i])`

**适用**：子集和、分割等和、完全背包（正序枚举即可）

---

## 模式三：区间 DP（合并/分割）

**状态**：`dp[i][j]` = 区间 `[i, j]` 上的最优值

**转移**：枚举分割点 k

```
dp[i][j] = min/max(dp[i][k] + dp[k+1][j] + cost(i,j))  for k in [i, j-1]
```

**枚举顺序**：按区间长度 `len` 从小到大

**适用**：矩阵链乘、戳气球、石子合并、回文分割

---

## 速记口诀

> **最后一步想选择，选择关联子问题，子问题定义即状态，选择遍历即转移。**
